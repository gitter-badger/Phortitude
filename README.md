# Phortitude

**Phortitude** is a collection of components for PHP developers.

It aims to comply with [PSR-1](http://www.php-fig.org/psr/psr-1), [PSR-2](http://www.php-fig.org/psr/psr-2) and
[PSR-4](http://www.php-fig.org/psr/psr-4).
